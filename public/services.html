Facebook In-Stream Ads Monetization လျှောက်ရန် လိုအပ်ချက်များ

Facebook In-Stream Ads Monetization ဘယ်လိုလျှောက်ရမလဲ၊ ဘယ်သူတွေ လျှောက်လို့ရလဲ? Facebook In-Stream Ads လျှောက်ဖို့ဆို အောက်ပါအချက်အလက်တွေနဲ့ ပြည့်စုံမှုရှိရပါမယ်။

  1. Facebook Page Follower ၅၀၀၀ အနည်းဆုံးရှိရပါမယ်။
  2. ရက် ၆၀ အတွင်း ဗီဒီယိုကြည့်ရှုမှု စုစုပေါင်းမိနစ် ၆၀၀၀၀ အနည်းဆုံးရှိရပါမယ်။
  3. Page မှာ ဗီဒီယိုအနည်းဆုံး ၅ ပုဒ်ရှိဖို့ လိုပါတယ်။
  4. ရှိတဲ့ဗီဒီယိုတွေကလည်း Copyright လွတ်ဖို့လိုပါမယ်။ သူများဆီက ပြန်ယူပြီး ပြန်တင်ထားတဲ့ ဗီဒီယိုတွေနဲ့ လျှောက်လို့မရပါဘူး။
  5. In-Stream Ads လျှောက်ဖို့အတွက် သတ်မှတ်ထားတဲ့ နိုင်ငံကနေဘဲ လျှောက်လို့ရပါမယ်။
  6. ဗီဒီယိုတွေကလည်း In-Stream Ads ထည့်ဖို့ခွင့်ပြုထားတဲ့ ဘာသာစကားနဲ့ ဖြစ်ရပါမယ်။

ဖော်ပြပါ အချက် ၆ချက်ထဲမှာ နံပတ်(၁) ကနေ (၄) အထိ ကိုက်ညီမှုရှိရင် In-Stream Ads အတွက် သတ်မှတ်ထားတဲ့ နိုင်ငံနဲ့ ဘာသာစကားကို ကျော်လွှားပြီး Monetization လုပ်လို့ရမှ ဖြစ်ပါတယ်။

Instream Ads လျှောက်ရန်မသင့်သောPageများ

  • Followers ငွေပေးပြီး တိုးထားသော page များ
  • Viewများကို ငွေနဲ့တိုးထားပြီး View ပြည့်လာသူများ
  • Pageသက်တမ်းတစ်လမပြည့်သေးသူများ
  • တလွဲဗီဒီယိုတွေနဲ့ View Follow ခိုးပြီး ISA set up ပေါ်နေသာpageများ
  • သူများvideoခိုးယူတင်သူများ
  • Copy Rightမလွတ်သော အသံသုံးသော video များ
  • Title မြန်မာလိုတက်ပြီး Country သေချာမရွေးပဲ ဖြစ်သလိုလုပ်ထားသောPageများ
  • မြန်မာနိုင်ငံ Location နဲ့ ISA set upလုပ်ဖူးသော pageများ
  • အလွန်အမင်း spam ဖြစ်အောင် Group များတွင် shareထား တင်ထားသော page များ စတဲ့ page တွေ ဖြစ်ပါသည်။

Monetization ဆိုင်ရာ ဝန်ဆောင်မှုများအား Lucky Charm ကုမ္ပဏီမှ ဆောင်ရွက်လျက်ရှိပြီး Monetization ဝန်ဆောင်မှုအပ်နှံလိုသော Client များအားလုံးတို့သည် ဖော်ပြပါအချက်များအား ကောင်းမွန်စွာ နားလည်သိရှိရမည်ဖြစ်သည်။

Facebook In-Stream Ads Monetization Service ဘယ်လိုလျှောက်မလဲ

  • Monetization ဝန်ဆောင်မှုကို အပ်နံလိုသော Client များတို့သည် မိမိတို့၏ Facebook Page ၌ Lucky Charm မှ Content Management Service ပေးမည့် သူအား Admin ခန့်ပေးရမည် ဖြစ်သည်။
  • Lucky Charm မှ ဝန်ဆောင်မှုပေးမည့်သူအား မိမိတို့၏ Facebook Page Admin ပေးပြီးသည်နှင့်တစ်ပြိုင်နက် Client များကိုယ်တိုင် Page Admin အဖြစ်မှ ထွက်ပေးရမည် ဖြစ်သည်။
  • Client များကိုယ်တိုင် Page Admin အဖြစ်မှ မထွက်ပါက Pre-Enrolled Page အဖြစ် Facebook တွင် စာရင်းဝင်သွားမည်ဖြစ်ပြီး Monetization လျှောက်၍ မရနိုင်ပါ။
  • မိမိတို့တင်စေချင်သော Content များအား Lucky Charm ဆီသို့ Google Drive ဖြင့်သော်လည်းကောင်း၊ အခြား Cloud Communication Tools များဖြင့်သော်လည်းကောင်း ပေးပို့ရမည်ဖြစ်ပြီး အဆိုပါ ပေးပို့ထားသော Content များအား Lucky Charm မှ Content Manager မှ တင်သွားပေးမည် ဖြစ်သည်။
